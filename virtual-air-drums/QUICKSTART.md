# 🚀 Quick Start Guide - Virtual Air Drums

## Step-by-Step Setup (5 minutes)

### 1️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 2️⃣ Generate Drum Sounds
```bash
pip install scipy
python setup_sounds.py
```

### 3️⃣ Run the Application
```bash
python air_drums.py
```

### 4️⃣ Start Playing!
- Position yourself 2-3 feet from the camera
- Raise your hands in front of the camera
- Make quick downward motions over the colored zones
- Press 'Q' to quit

## 🎯 Drum Zones

| Zone | Color | Position | Sound |
|------|-------|----------|-------|
| Hi-Hat | Red | Top Left | High pitched |
| Snare | Green | Top Right | Sharp snap |
| Bass | Blue | Bottom | Deep thump |

## 💡 Tips for Best Experience

1. **Lighting** - Ensure good lighting on your hands
2. **Background** - Use a simple, uncluttered background
3. **Speed** - Quick downward motions work best
4. **Volume** - Faster hits = louder sounds
5. **Position** - Keep hands clearly visible to camera

## ⚡ Keyboard Controls

- **Q** - Quit application
- **ESC** - Alternative quit

## 🔧 Common Issues

**Camera not working?**
- Check if another app is using the camera
- Grant camera permissions

**No sound?**
- Check volume is turned up
- Verify sound files exist in `sounds/` folder

**Hands not detected?**
- Improve lighting
- Move closer to camera
- Keep hands in frame

## 🎵 Enjoy Drumming!

Have fun creating beats with your virtual air drums! 🥁
