WHAT HAPPENS WHEN YOU PLAY:
======================================================================

EVENT 1: You raise your right hand
─────────────────────────────────────
  → Green skeleton lines appear on your hand
  → Yellow circle appears on your index finger
  → Circle follows your finger in real-time

EVENT 2: You move hand over RED zone (Hi-Hat)
─────────────────────────────────────
  → Yellow circle is now inside the red zone
  → Zone stays semi-transparent (waiting for hit)

EVENT 3: You make a QUICK DOWNWARD motion
─────────────────────────────────────
  → System detects: velocity = 450 px/s
  → Velocity > 300 threshold ✓
  → Downward motion ✓
  → Inside zone ✓
  → Cooldown OK ✓
  → HIT REGISTERED!

EVENT 4: Visual and audio feedback
─────────────────────────────────────
  → Red zone FLASHES BRIGHT for 0.2 seconds
  → Sound plays: "Tss!" (Hi-hat sound)
  → Volume: 0.23 (based on velocity)
  → Latency: ~8ms

EVENT 5: You move to GREEN zone (Snare)
─────────────────────────────────────
  → Quick downward motion
  → Green zone FLASHES BRIGHT
  → Sound plays: "Crack!" (Snare sound)
  → Volume: 0.41 (faster hit = louder)

EVENT 6: You use BOTH hands on BLUE zone (Bass)
─────────────────────────────────────
  → Both hands detected simultaneously
  → Quick downward motion
  → Blue zone FLASHES BRIGHT
  → Sound plays: "BOOM!" (Bass drum sound)
  → Volume: 0.60 (very fast hit = loud)

======================================================================
TIPS FOR BEST EXPERIENCE:
======================================================================

✅ DO:
  • Sit 2-3 feet from webcam
  • Use good lighting
  • Make quick, deliberate downward motions
  • Think "drumstick hitting drum"
  • Experiment with different speeds
  • Use both hands for complex beats

❌ DON'T:
  • Move too slowly (won't trigger)
  • Just hover without moving down
  • Play in the dark
  • Cover the camera
  • Move too far from camera


======================================================================
TROUBLESHOOTING:
======================================================================

If hands not detected:
  → Improve lighting
  → Move closer to camera
  → Keep hands clearly visible

If hits not registering:
  → Move FASTER (quick downward motions)
  → Make sure you're moving DOWN, not sideways
  → Check you're inside the colored zone

If no sound:
  → Check volume is turned up
  → Verify sound files exist in sounds/ folder
  → Run: python setup_sounds.py


======================================================================
TO RUN THE ACTUAL APPLICATION:
======================================================================

Step 1: Install Python
  • Download from python.org or Microsoft Store
  • Make sure "Add to PATH" is checked

Step 2: Install Dependencies
  python -m pip install opencv-python mediapipe pygame numpy scipy

Step 3: Generate Sounds
  python setup_sounds.py

Step 4: Run Application
  python air_drums.py

Step 5: Start Playing!
  • Raise your hands
  • Make drumming motions
  • Have fun! 🎵
