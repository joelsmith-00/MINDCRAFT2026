🔥 Firebase Setup Wizard

DOTDEV Club App - Step-by-Step Guide

1
Create Firebase Project

Let's create your Firebase project in just a few clicks!

  1. Click the button below to open Firebase Console
  2. Sign in with your Google account
  3. Click "Add project"
  4. Enter project name: dotdev-club
  5. Disable Google Analytics (optional for now)
  6. Click "Create project"
🚀 Open Firebase Console
2
Register Android App

Now let's register your Android app with Firebase.

  1. In Firebase Console, click the Android icon
  2. Enter the package name below (click to copy):
com.dotdev.dotdev_club
  1. Click "Register app"
  2. Download google-services.json
  3. Place it in: android/app/google-services.json
  4. Click "Next" through the remaining steps
⚠️ Important: Make sure the file is named exactly google-services.json and is placed in the android/app/ folder!
3
Enable Authentication

Enable email/password authentication for your users.

  1. In Firebase Console sidebar, click "Authentication"
  2. Click "Get started"
  3. Click the "Sign-in method" tab
  4. Click on "Email/Password"
  5. Toggle "Enable"
  6. Click "Save"
4
Create Firestore Database

Set up your cloud database to store app data.

  1. In Firebase Console sidebar, click "Firestore Database"
  2. Click "Create database"
  3. Select "Start in production mode"
  4. Choose a location closest to you (e.g., asia-south1 for India)
  5. Click "Enable"
5
Update Firestore Security Rules

Configure security rules to protect your data.

  1. In Firestore Database, click the "Rules" tab
  2. Replace the existing rules with the code below:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }
  1. Click "Publish"
6
Enable Cloud Storage

Enable file storage for project uploads.

  1. In Firebase Console sidebar, click "Storage"
  2. Click "Get started"
  3. Start in "production mode"
  4. Use the same location as Firestore
  5. Click "Done"
  6. Go to "Rules" tab and replace with:
rules_version = '2'; service firebase.storage { match /b/{bucket}/o { match /{allPaths=**} { allow read, write: if request.auth != null; } } }
  1. Click "Publish"
7
Get Firebase Configuration

Retrieve your Firebase configuration values.

  1. Click the gear icon ⚙️ in Firebase Console sidebar
  2. Click "Project settings"
  3. Scroll down to "Your apps" section
  4. Find your Android app
  5. Look for "SDK setup and configuration"
  6. Copy the values and paste them below:

8
Run Your App!

You're almost there! Let's run your app.

  1. Open your terminal/command prompt
  2. Navigate to your project folder
  3. Run the following command:
flutter run
🎉 Congratulations! Your DOTDEV Club app should now be running with Firebase integration!
